Une API de contrepèteries en français !


Présentation

De quoi s'agit-il ?

Nous sommes une API de contrepèteries francophones proposant plus de 600 contrepèteries classées par catégorie et par niveau de vulgarité.


D'où viennent les contrepèteries ?

Les contrepèteries viennent en grande partie du site La poule qui mue, de l'ouvrage de Louis Perceau La Redoute des contrepèteries et de l'application Contrepèterie Quizz. Le reste vient de recherches, de nous et de notre entourage.

La contrepèterie est un art grivois, vous pourriez vous retrouver face à des termes choquants ou en montrer à des personnes qui ne le souhaitent pas.


Documentation

Utilisation

Pour utiliser l'API Rest, vous n'avez qu'à faire comme suit. Vous recevrez une contrepèterie aléatoire, par id, par catégorie ou par niveau de vulgarité en fonction de votre choix.


Aléatoire 
GET /api/random

Réponse attendue 
{
    "phrase": "Le docteur Barnard se hisse sur la porte.",
    "indice_MD": "Le docteur Barnard se **h**isse sur la **p**orte.",
    "indice_HTML": "Le docteur Barnard se <b>h</b>isse sur la <b>p</b>orte.",
    "reponse": "Le docteur Barnard se pisse sur l'ahorte.",
    "categorie": "Médecine",
    "id": 3,
    "vulgarite": ["1/5", "20%"];
}


Par catégorie 
GET /api/random?cat=lieux

Réponse attendue 
{
    "phrase": "Regarde, on voit les Pouilles du cap !",
    "indice_MD": "Regarde, on voit les **P**ouilles du **c**ap !",
    "indice_HTML": "Regarde, on voit les <b>P</b>ouilles du <b>c</b>ap !",
    "reponse": "Regarde, on voit les couilles du Pape !"
    "categorie": "Lieux",
    "id": 58,
    "vulgarite": ["3/5", "60%"];
}

Par ID 
GET /api/random?id=34

Réponse attendue 
{
    "phrase": "L'Afrique se développe dans le troc.",
    "indice_MD": "L'A**f**rique se développe dans le **t**roc.",
    "indice_HTML": "L'A<b>f</b>rique se développe dans le <b>t</b>roc.",
    "reponse": "La trique se développe dans le froc."
    "categorie": "Lieux",
    "id": 34,
    "vulgarite": ["2/5", "40%"];
}

Il vous est évidemment possible de combiner plusieurs filtres.
